3. Після запуску — `ADMIN_GUIDE.md` як путівник по адмінці

`LOGO_BRIEF.md` — самодостатній brief. Можна:
- Прогнати промпти через AI (Gemini, Midjourney, Ideogram)

## Ключові архітектурні рішення

### Технологічний стек

- **Frontend:** Next.js 15 App Router + TypeScript + Tailwind v4 + shadcn/ui + Magic UI + Framer Motion + Lenis
- **Backend / CMS:** Payload CMS 3.x (open source, embedded в Next.js)
- **БД:** PostgreSQL 16
- **Інфра:** Vultr VPS (2 vCPU / 4 GB / 80 GB SSD, Ubuntu 24, Frankfurt) + Docker + Nginx + Let's Encrypt

### AI-стек

- **AI-Concierge Лія:** Claude API (Anthropic) — auto-routing Haiku/Sonnet/Opus
- **AI-content (блог, картки):** Claude API
- **AI-фото:** Google Gemini API (Nano Banana — image generation і edit)
- **Voice transcription:** Whisper open source локально на VPS

### Канали комунікації

- Сайт-чат
- Telegram-бот
- Viber-бот
- ❌ БЕЗ email
- ❌ БЕЗ Instagram

### Платежі

- Monobank Acquiring (основний)
- LiqPay (резервний)
- ПРРО Checkbox.ua (фіскальні чеки в Telegram/Viber)
- 100% передоплата ОЛЛО АБО часткова передоплата % при доставці кур'єру

### Доставка

- **Вручну Варварою**, без API кур'єрських сервісів
- Зони редаговані в адмінці (Ірпінь-Буча-Гостомель + Київ)
- Святковий режим, аварійний режим

## Ключові не-стандартні рішення

1. **AI-Concierge повністю обробляє замовлення** — від консультації до оплати, до Варвари йде вже оплачений заказ
2. **AI-фото pipeline як продуктова перевага** — масштабування каталогу за хвилини
3. **Без email** — комунікація тільки в месенджерах
4. **Без cookie banner** — архітектура без third-party tracking
5. **Самописна server-side analytics** — анонімізована, через хеш IP
6. **Знижки з таймером per-bouquet** — без програм лояльності, без промокодів
7. **Демо-режим як first-class feature** — Варвара починає не з пустого сайту
8. **30 SEO-блог-статей перед запуском** — запуск з вже розкрученим контентом
9. **Editorial mobile-first ефекти** — performance budget Lighthouse Mobile ≥85
