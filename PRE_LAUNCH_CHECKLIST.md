## 🔥 Критичні (зробити в перший тиждень — інакше затримаємо запуск)

### 1. Trademark check + реєстрація «Florenza»

**Що зробити:**
1. Перейти на https://base.uipv.org (Укрпатент)
2. Знайти розділ «Торговельні марки»
3. Перевірити доступність назви «Florenza» в класах:
   - **Клас 31** — живі рослини, квіти
   - **Клас 35** — реклама, послуги торгівлі
   - **Клас 39** — пакування і доставка товарів
4. Якщо назва вільна — **рекомендую одразу подати заявку на реєстрацію**

**Час:** перевірка 30 хвилин, реєстрація ~9 місяців (паралельно з усім).

**Бюджет:**
- Самостійно через Укрпатент: ~3000 грн збір
- Через юриста: 4500–7000 грн (рекомендовано — ризик помилок менший)

**Якщо назва зайнята:** ⚠️ Необхідно змінити імʼя бренда **ДО початку розробки сайту**. У такому випадку — звʼязатися з підрядником, дати інші 3–5 варіантів імен (можу запропонувати).

### 3. Зареєструвати VPS на Vultr

**Що:**
1. Створи акаунт на https://vultr.com
2. Поповни баланс на $20–30
3. Створи Cloud Compute інстанс:
   - **Plan:** $12/міс (2 vCPU / 4 GB RAM / 80 GB SSD)
   - Можна почати з $6/plan і апгрейдити пізніше
   - **Region:** Frankfurt (DE) — найближчий до України, низька латентність
   - **OS:** Ubuntu 24.04 LTS x64
   - **Hostname:** `florenza`
   - **SSH key:** ⚠️ ОБОВ'ЯЗКОВО додай SSH-ключ (не пароль!), для безпеки

**Час:** 30 хвилин

**Бюджет:** $6–12/міс залежно від плану

**Після створення:**
- Запиши IP-адресу VPS
- Передай підряднику IP + SSH-ключ-доступ

### 4. Налаштувати DNS на GoDaddy → Vultr

**Що зробити:**
1. У GoDaddy → твій домен → DNS Settings
2. Видалити дефолтні `A`/`CNAME` записи що ведуть на парковку
3. Додати нові:
   - `A` record: `@` → `<IP твого VPS>`
   - `A` record: `www` → `<IP твого VPS>`
   - `A` record: `admin` → `<IP твого VPS>` (опційно — для адмінки на піддомені)

**Час:** 15 хвилин

**Розповсюдження:** до 24 годин (зазвичай 1–2 години)

## 🟡 Важливі (зробити в перший-другий тиждень)

### 5. Подати заявку на Monobank Acquiring

**Що:** Договір на прийом онлайн-оплат через Mono.

**Як:**
1. Перейти на https://api.monobank.ua/docs (для розробників) або https://acquire.monobank.ua
2. Натиснути «Підключити еквайринг»
3. Заповнити форму: ФОП Каракой Варвара Олександрівна, тип діяльності, оборот
4. Дочекатись зворотного звʼязку від менеджера Mono (1–3 робочих дні)
5. Підписати договір електронно через Дію
6. Пройти KYC (надати скан паспорта і виписки ФОП)
7. Отримати API-ключі (Token + Public key)

**Час від заявки до отримання ключів:** 1–2 тижні

**Бюджет:** без абонплати, тільки комісія 1.4–2% з кожної оплати

**Передати підряднику:**
- Mono API Token
- Mono Public Key

### 7. Підключити ПРРО (Програмний РРО) — Checkbox.ua

**Що:** Обовʼязково за UA-законом для онлайн-оплат. Видає фіскальні чеки.

**Чому Checkbox.ua:**
- Лідер UA-ринку для малого бізнесу
- Інтегрується з Mono і LiqPay через webhook
- **Підтримує відправку чеків в Telegram/Viber через API** — критично, бо ми не використовуємо email

**Як підключити:**
1. Перейти на https://checkbox.ua
2. Зареєструватись на ФОП
3. Підключити каси: вкажіть РРО з податкової
4. Активувати API:
   - Створити робоче місце через адмінку Checkbox
   - Отримати `LICENSE_KEY` для API
   - Отримати `OPERATOR_PIN` (PIN-код касира)
5. Налаштувати інтеграцію з Monobank через панель Checkbox

**Час:** 1–2 дні

**Бюджет:** ~250–500 грн/міс залежно від обороту

**Передати підряднику:**
- Checkbox LICENSE_KEY
- Checkbox OPERATOR_PIN
- Checkbox CASH_REGISTER_ID

**Альтернативи (якщо Checkbox не підходить):**
- Vchasno.Каса
- ПРРО ПриватБанку (тільки якщо платить через LiqPay)

### 8. Зареєструвати Telegram-бот

**Що:** Бот через якого спілкується Лія в Telegram.

**Як:**
1. У Telegram знайти @BotFather
2. Команда `/newbot`
3. Імʼя бота: `Florenza | Бутік квітів`
4. Username бота: `@florenza_bot` (або найближчий доступний — `@florenza_irpin_bot`, `@florenza_kvity_bot`)
5. Зберегти **Token** (виглядає як `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
6. Налаштувати бота:
   - `/setdescription` → *«AI-консультантка Florenza. Допоможу обрати букет та оформити замовлення з доставкою по Ірпеню, Бучі, Гостомелю»*
   - `/setabouttext` → коротка версія
   - `/setuserpic` → монограма F (з логотипа)
   - `/setcommands` → налаштувати список команд:
     ```
     start - Почати спілкування з Лією
     katalog - Подивитися каталог букетів
     dostavka - Дізнатися про доставку
     stop - Відписатися від розсилок
     help - Допомога
     ```

**Час:** 10 хвилин

**Бюджет:** безкоштовно

**Передати підряднику:**
- Telegram Bot Token
- Bot username

### 15. Опціонально: підготувати фото

**Що:** 3–5 фото реальних букетів на телефон для seed AI-генератора.

**Чому:** AI-фото буде стабільнішим, якщо є реальні референси «що ми взагалі можемо зробити». Без референсів AI генерує «букети взагалі», з референсами — «букети Florenza».

**Як зробити:**
1. Зібрати/купити 3–5 типових букетів
2. Сфотографувати на нейтральному фоні (білому, бежевому, лляному):
   - Максимально природне світло (вікно)
   - Без використання спалаху
   - 3 ракурси кожного букета: фронт, кут 45°, top-down
3. Зберегти в папку `bouquets-references/`
4. Передати підряднику для seed калібровки

**Час:** 1 година на 5 букетів

**Бюджет:** ~2000–5000 грн на покупку 5 букетів-моделей

### 16. Записати голосове інтервʼю для About-сторінки

**Що:** AI створить заглушку About, але справжня історія Варвари — особиста.

**Як:**
1. Знайти 20 хвилин тиші
2. Записати голосове повідомлення (на телефон) відповідаючи на питання:
   - Звідки любов до квітів? Як прийшла в флористику?
   - Що для тебе означає Ірпінь? Чому магазин саме тут?
   - Що відрізняє Florenza від інших флористів?
   - Що ти хочеш щоб клієнт відчув коли отримує наш букет?
   - Як ти бачиш флористику як ремесло, а не «продаж квітів»?
3. Передати голосове підряднику
4. AI транскрибує + перетворює на editorial-сторітелінг
5. Варвара ревьюить і доводить вручну

**Час:** 20 хвилин запису + 1 година ревізія тексту
